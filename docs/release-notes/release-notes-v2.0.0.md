# Pyserini Release Notes (v2.0.0)

+ **Release date:** April 19, 2026
+ **Anserini dependency:** v2.0.0
+ **Lucene dependency:** v10.4.0

## Summary of Changes

+ Known issues are [enumerated separately](./known-issues-v2.0.0.md).
+ Updated to the Anserini v2.0.0 fatjar, which uses Lucene 10.4.0.
+ Raised the Python baseline to 3.12 and refreshed package dependencies.
+ Refined M-BEIR 2CR support with an updated YAML template.
