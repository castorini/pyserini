# Pyserini Release Notes (v1.4.0)

+ **Release date:** December 16, 2025
+ **Anserini dependency:** v1.4.0
+ **Lucene dependency:** v9.9.1

## Summary of Changes

+ Updated 2CR for M-BEIR (modified metrics, added query instructions).
+ Propagated changes from [anserini#3037](https://github.com/castorini/anserini/pull/3037) affecting BEIR arguana.
+ Aligned fusion methods between Anserini + Pyserini.
+ Added corpus encoding for SPLADE.
+ Pulled in latest index symbols from Anserini.
+ Fixed `judged.k` metric in `trec_eval`.
