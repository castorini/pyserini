# Pyserini Release Notes (v2.3.0)

+ **Release date:** June 14, 2026
+ **Anserini dependency:** v2.2.0
+ **Lucene dependency:** v10.4.0

## Summary of Changes

+ Updated to the Anserini v2.2.0 fatjar.
+ Fixed UniIR compatibility with Transformers 5.
+ Hardened prebuilt index URL checks and added tests for generated prebuilt index documentation.
+ Updated MIRACL and Mr.TyDi Lucene index metadata.
+ Updated mBEIR 2CR scores and documentation.
+ Updated BEIR SPLADE++ 2CR commands to encode queries on the fly instead of using cached queries.
+ Removed obsolete BEIR SPLADE-distil and SPLADE++ topic aliases.
+ Confirmed following 2CRs all pass: `v1-passage`, `v1-doc`, `v2-passage`, `v2-doc`, `beir`, `bright`, `miracl`, `mrtydi`, `ciral`, `odqa`, `atomic`, `m_beir`.
