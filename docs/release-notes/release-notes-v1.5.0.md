# Pyserini Release Notes (v1.5.0)

+ **Release date:** January 14, 2026
+ **Anserini dependency:** v1.5.0
+ **Lucene dependency:** v9.9.1

## Summary of Changes

+ Updated to Anserini v1.5.0 fatjar to align with changes.
+ Upgraded dependent libs.
+ Added M-BEIR topics and qrels.
+ Added support for image datasets and MCP bindings.
+ Fixed REST API compatibility issues and refactored MCP server.
