# Pyserini Release Notes (v1.6.0)

+ **Release date:** April 6, 2026
+ **Anserini dependency:** v1.7.1
+ **Lucene dependency:** v9.9.1

## Summary of Changes

+ Updated to Anserini v1.7.1 fatjar and fixed compatibility issues with the new release.
+ Expanded support for multimodal retrieval benchmarks: M-BEIR, MMEB, and DSE.
+ Expanded support for BRIGHT retrieval benchmark.
+ Added and refined MCP functionality with a new client, server updates, image URL support, and cleaner tooling.
+ Improved encoder and retrieval support with Qwen3 updates, Faiss GPU batch query encoding, and related fixes.
+ Hardened dataset/query handling with iterator checks, automatic downloads for M-BEIR assets, and additional tests.
