# Pyserini - Unified Retrieval Architecture (URA) Extension Lab

This repository is a fork of the official Pyserini infrastructure, utilized to deploy and benchmark advanced features of the Unified Retrieval Architecture (URA) framework.

## Latest Updates: Phase 3 Evaluation Pipeline
- **Root-Level Implementation:** Dedicated integration notebook `URA_Baseline_Pipeline.ipynb` is hosted directly at the root directory for maximum accessibility and deployment efficiency.
- **Automated Validation:** Features a self-contained execution pipeline that provisions OpenJDK 21, resolves environmental JNI paths, and runs native Lucene searching.
- **Empirical Benchmarking:** Implements a simulated Lexical (BM25) and Dense Semantic fusion scoring matrix verified against standardized evaluation targets.

## Quick Start
To review or execute the baseline validation architecture, open `URA_Baseline_Pipeline.ipynb` directly from the root level of this repository inside a containerized runtime environment.
